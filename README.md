## 🚀 Kom i Gang

1. **Klone prosjektet**
```bash
git clone https://github.com/Marcus-Kodehode/my-awesome-cv.git
cd my-awesome-cv
```

2. **Installer avhengigheter**
```bash
npm install
```

3. **Start utviklingsserver**
```bash
npm run dev
```

4. **Åpne nettleseren**
```
http://localhost:3000
```
